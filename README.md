# 🏆 Real Acade - Gestión de Club

¡Tu aplicación está lista para el mundo real! Sigue estos pasos para obtener tu link sin tener que instalar nada en tu computadora.

## 🚀 Cómo obtener tu Link (Paso a Paso)

### 1. Sube tu código a GitHub (El "Almacén")
*   Crea un nuevo repositorio en tu cuenta de GitHub (ponle de nombre `real-acade`).
*   En la página principal de tu repositorio vacío, busca el enlace que dice **"uploading an existing file"**.
*   Arrastra todos los archivos de este proyecto a esa ventana. **Importante**: Asegúrate de subir todas las carpetas (`src`, `docs`, `components`, etc.) y los archivos sueltos (`package.json`, `next.config.ts`, etc.).
*   Haz clic en el botón verde **"Commit changes"** al final de la página.

### 2. Conecta con Firebase App Hosting (El "Servidor")
*   Ve a la [Consola de Firebase](https://console.firebase.google.com/).
*   Selecciona tu proyecto: `studio-8891359241-1a978`.
*   En el menú izquierdo, ve a **App Hosting** y dale a "Comenzar".
*   Conecta tu cuenta de GitHub y selecciona el repositorio `real-acade` que acabas de crear.
*   En la configuración de despliegue, deja todo como viene por defecto.
*   **Facturación**: Si te pide actualizar el plan, selecciona el **Plan Blaze**. Recuerda que para un club de amigos, el costo será **$0.00** gracias a los límites gratuitos de Google Cloud.

### 3. ¡Obtén tu URL!
*   Al terminar el despliegue (tarda unos 2-3 minutos), Firebase te dará una URL pública (ej: `https://real-acade.web.app`).
*   ¡Esa es la que debes pasar por el grupo de WhatsApp!

## 💡 Preguntas Frecuentes
*   **¿Es seguro el Plan Blaze?**: Sí, Google te regala un uso muy amplio cada mes. Tendrías que tener miles de visitas al día para que te cobren algo.
*   **¿Cómo actualizo la app?**: Cada vez que quieras cambiar algo, solo tienes que subir los archivos nuevos a GitHub. Firebase se dará cuenta y actualizará el link automáticamente.
*   **¿Puedo borrarlo?**: Sí, puedes volver al plan gratuito o borrar el proyecto en cualquier momento desde la configuración de Firebase.
