# Real Acade - Gestión de Club

¡Bienvenido a la plataforma oficial de Real Acade! Esta aplicación está diseñada para gestionar las estadísticas de tus partidos de Fútbol 7 de forma profesional.

## 🚀 Instrucciones para el Despliegue Final

Para que tu aplicación esté activa y puedas compartir el link con tus amigos, sigue estos pasos:

1. **Firebase Console**: Ve a la [Consola de Firebase](https://console.firebase.google.com/).
2. **Habilitar Servicios**:
   - En **Authentication**, activa el método de "Correo electrónico/contraseña".
   - En **Firestore Database**, asegúrate de que la base de datos esté creada en "Modo de producción".
3. **Despliegue (Deploy)**:
   - Utiliza la herramienta de **Firebase App Hosting** o **Hosting** dentro de la consola para conectar este repositorio y generar tu URL pública (`https://real-acade.web.app` o similar).
4. **Primer Inicio**:
   - Una vez desplegada, entra al link, regístrate y pulsa el botón **"Activar Modo Administrador"** en el Panel de Control. ¡Ese botón desaparecerá una vez que seas admin!

## 🛠️ Funcionalidades Incluidas
- **Panel de Control**: Estadísticas globales y Power Ranking.
- **Historial de Partidos**: Crónicas generadas por IA y votación de mejores goles.
- **Gestión de Jugadores**: Perfiles detallados con historial de rendimiento.
- **Seguridad**: Solo el administrador puede editar datos.

¡Disfruta gestionando tu club!
