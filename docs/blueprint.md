# **App Name**: Fut7Stats

## Core Features:

- Match Data Upload: Allow users to upload match data (goals, assists, fouls, etc.)
- Data Visualization: Generate charts and graphs to visualize team and player statistics. For example: goals scored per game, assists per player, etc.
- Player Profiles: Create individual player profiles to track their performance over time. Show total goals, assists, and other key stats.
- Ranking Generation: Automatically calculate and display rankings based on match data (e.g., top scorer, best assistant, least fouls, etc.).
- Match History: Maintain a historical record of all matches played, including key statistics and outcomes. Each event will be indexed.
- User Authentication: Require each player to create a user account to log in.
- Player Performance Rating: Allow each user to rate the performance of every player in the match from 1 to 10 after the match is finished.

## Style Guidelines:

- Primary color: Vibrant green (#50FA7B) to evoke the energy and excitement of the sport.
- Background color: Dark grey (#282A36) to provide contrast and focus on the content.
- Accent color: Electric blue (#8BE9FD) to highlight interactive elements and important data.
- Body and headline font: 'Inter', a sans-serif font providing a clean, modern, objective look that is suitable for data visualization.
- Use sports-themed icons to represent different statistics and actions.
- Dashboard layout with clear sections for match data, player profiles, and rankings.
- Subtle transitions and animations when loading and displaying data.
